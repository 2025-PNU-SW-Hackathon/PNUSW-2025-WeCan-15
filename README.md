# WeCam 프로젝트 소개

## 1. 프로젝트 개요

### 1.1. 개발 배경 및 필요성

현재 대학생 자치조직(학생회) 운영은 여전히 **비효율적인 메신저 중심 체계**에 머물러 있으며,  
업무 인수인계, 공지사항 전달, 일정 관리 등에서 **전용 플랫폼이 부재**한 상황입니다.

WeCam은 단순한 협업툴이 아니라  
**"대학생활에 필수적인 플랫폼"**으로서  
학생회 운영 효율화와 학생과의 소통 강화를 통해  
**투명한 조직 운영과 지속 가능한 구조**를 지원합니다.

---

### 1.2. 개발 목표 및 주요 내용

* **목표**  
  → 학생회 중심의 **체계적인 대학 플랫폼 구축**

* **주요 기능**
  * 학생회 조직별 워크스페이스 제공
  * 일정 및 업무 관리 기능
  * 인수인계 기능 내장
  * 공지사항 공유 (학생회 ↔ 일반 학생)
  * 자동 채팅방 생성 및 알림 기능
  * 학생 개별 참여 기능 (캘린더/공지/설문)

---

### 1.3. 주요 설계 요소

* **조직 구조**  
  → 총학생회 → 단과대학 학생회 → 학과 학생회  
    (계층형 워크스페이스 구성)

* **가입 및 승인**
  * 학생회는 승인 후 워크스페이스 생성
  * 일반 학생은 학교/학과 선택 후 가입

* **핵심 기능**
  * 일정 관리 (캘린더 기반)
  * 업무 인수인계 게시판
  * 공지사항/설문 기능
  * 실시간 알림 및 채팅

---

### 1.4. 기존 서비스 대비 차별성

* **학생회 중심 설계** (기존 협업툴과 차별화)
* **학교 조직 체계 자동 구성**
* **학생 대상 공지/채팅 연동**
* **학생회 교체 시 인수인계 기능 내장**
* **학생 개인 사용자에게도 유용한 캠퍼스 플랫폼 지향**

---

### 1.5. 사회적 가치 및 기대효과

* 학내 **커뮤니티 활성화**
* 학생회의 **투명한 운영 문화** 조성
* **업무 자동화**로 학생회 운영 부담 경감
* 일반 학생의 **참여 확대** (공지, 알림, 설문)
* 장기적으로 **대학생 필수 앱**으로 자리매김

### 3.2. 기능 설명

#### 로그인

* 이메일/비밀번호 입력
* 유효성 검사 후 로그인 가능
* 성공 시 대시보드 이동
* 실패 시 에러 메시지 표시

#### 워크스페이스 생성

* 학생회 승인 프로세스 포함  
  (초대코드 기반 자동 승인 / 수동 승인)
* 계층형 조직 구조 자동 구성

#### 일정 관리

* 일정 추가 시 알림 자동 발송
* 캘린더 뷰 제공

#### 공지사항

* 학생회 → 일반 학생 공지 기능
* 열람 여부 확인 가능

#### 업무 인수인계

* 게시판 형태로 운영
* 이전 학생회 기수와 연동

#### 채팅

* 조직별 자동 채팅방 생성
* 실시간 메시지 전송 및 알림

## 4. 설치 및 사용 방법

### 설치

* Docker / Docker Compose 설치
* Node.js / npm 설치

### 실행 방법

#### Backend

bash
./gradlew build
java -jar build/libs/wecam-backend.jar


#### Frontend

bash
npm install
npm start


#### Docker Compose (전체 실행 시)

bash
docker-compose up --build
